                    huggingOs - SIMPLE START GUIDE
================================================================================

WHAT YOU NEED:
1. Ubuntu/WSL installed (you have it!)
2. Build tools installed
3. VirtualBox installed (you have it!)

QUICK START:

1. Install build tools (if not done):
   ./INSTALL_TOOLS.sh

2. Build the OS:
   ./quick_build.sh

3. Run in VirtualBox:
   - Open VirtualBox
   - Create new VM (Type: Other, Version: Other/Unknown 32-bit)
   - Memory: 128 MB
   - In Storage settings, attach huggingOs.iso
   - Start the VM!

NEW FEATURES ADDED:
- color command - Change terminal colors
- calc command - Simple calculator
- banner command - Show ASCII art
- about command - OS information
- Enhanced welcome screen
- Better error handling

AVAILABLE COMMANDS:
help, clear, info, version, echo, reboot, color, calc, banner, about
