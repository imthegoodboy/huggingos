# How to Run huggingOs

## Quick Start

1. **Install tools** (first time only):
   ```bash
   ./INSTALL_TOOLS.sh
   ```

2. **Build the OS**:
   ```bash
   ./quick_build.sh
   ```

3. **Run in VirtualBox**:
   - Create new VM (32-bit, Other/Unknown)
   - Attach `huggingOs.iso` in Storage
   - Start VM

## New Commands Available

- `color <name>` - Change terminal color (green, cyan, yellow, red, blue, magenta, white)
- `calc <num1> <op> <num2>` - Calculator (e.g., `calc 5 + 3`)
- `banner` - Show ASCII art banner
- `about` - About huggingOs
- `help` - Show all commands
